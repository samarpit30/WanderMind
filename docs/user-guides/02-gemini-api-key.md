# Gemini API Key Setup Guide

This guide walks you through obtaining your Gemini API key from Google AI Studio.

## Steps

### 1. Sign in to Google AI Studio
1. Go to [Google AI Studio](https://aistudio.google.com/).
2. Log in using your Google Account.

### 2. Create the API Key
1. Click the blue **Get API key** button in the top left or center panel.
2. Select **Create API key**.
3. Choose either **Create API key in new project** or **Create API key in existing project** (you can use the same GCP project created for Google Maps).
4. Copy the generated API key.

### 3. Add Key to Environment Variables
Open your local `.env.local` file and paste the key under `GEMINI_API_KEY`:
```env
GEMINI_API_KEY=your_gemini_api_key_here
```

## ✅ How to know it worked
Open a terminal and run the following curl command (replacing `YOUR_GEMINI_API_KEY` with your actual key) to test model access:

```bash
curl -H "Content-Type: application/json" \
     -d '{"contents":[{"parts":[{"text":"Hello, Gemini!"}]}]}' \
     "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=YOUR_GEMINI_API_KEY"
```

If it succeeds, it will return a JSON response containing a text reply generated by the Gemini model.
